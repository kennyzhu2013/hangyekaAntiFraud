# 目的

重庆行业卡主要用于拨打银行贷款营销、贷款催收、零售推销、房产中介等电话；为确保行业卡合法合规使用，需根据通话录音转写的ASR文本，识别该通话是否高风险涉诈；本规范旨在明确重庆反诈语音质检规则的复核规范要求，以便对基于AI模型的通话质检结果进行人工二次确认，确保通话符合业务安全要求。

# 原则

- 准确性：复核内容应准确反映语音沟通中的实际信息和意图。

- 一致性：对于相同的语音内容，应采用统一的复核方式，确保复核结果的一致性。

- 完整性：复核应涵盖语音沟通中的关键信息，包括但不限于问题类型、涉及主体、相关细节等。

# 复核规范

正常/违规场景判断 + 风险等级判断 + 判断说明

- **正常场景：**正常场景直接判断正常提交即可


- **违规场景：**对于违规场景，需判断------风险等级（按违规场景识别与涉诈识别中风险等级判定）+判断说明（按违规/涉诈类型+分析规范填写）如下：

判定说明需按照【违规/涉诈类型+分析】的规范，分析需简要概括通话的具体违规行为

如：企业营销与招商服务：ai推广获客服务

贷款相关；拍拍贷平台贷款推销

引流第三方平台：贷款推销，引导添加放款经理微信

证券投资类：引导参与抱团股票投资，签合同，涉诈

个体工商户年报补录收费：自称线上税务申报代办专员，引导通过服务号操作税务申报，缴纳补录费用，涉诈


# 安全策略

## 违规场景识别

针对每日AI质检结果异常通话，需根据通话内容判断是否存在违规场景；对确认违规通话，需进一步判断风险等级与复核说明总结；

对违规场景的判断，采用**就高不就低**的原则进行分类，如通话中引导微信群【引流第三方平台】，明确为炒股群，因【证券投资类】风险级别更高，可判断为【证券投资类】。

### 贷款相关

1.  违规场景说明

- 贷款债务优化（高风险）：如帮处理高利息平台债务、以法律咨询名义诱导做贷款替换（借低息贷款还高息贷款）

- 砍头息贷款（高风险）：如贷款1w到手8k

- ab贷（高风险）：受托支付、委托支付、第三方代收、第三方到场背调、第三方紧急联系人等

- 过分承诺（低风险）：包过、包批、100%下款、内部渠道、保证金、解冻金、零利息；

- 提前收取费用（低风险）：定金

- 网贷、小贷平台（低风险）：度小满、安逸花、拍拍贷、分期乐、理想花、花呗等

- 不看征信（低风险）：贷款推销不看征信

- 合规平台（合规）：银行、银行消费金融、融三六零贷款

- 助贷包装（合规）：个人贷款包装为经营性贷款

2.  判断方法

    凡是提前交费、包下款、非银行合作高利息网贷、套路贷等，均为违规。

### **手机租赁套路贷诈骗**

1.  诈骗套路说明

## 违规场景涉诈判断与补充

### **场景补充：**

如发现通话中存在其他违规场景，可根据具体场景进行复核标注，对于新增类型违规场景，按以上同类型场景进行风险等级判定；如不属于上述类型违规/涉诈场景，可判定为其他。

### **违规场景涉诈判断：**

如发现违规场景中部分类型疑似涉诈，可按如下流程做进一步判断。

（1）通过多个同类型疑似涉诈的违规通话综合判断；

（2）对通话中提到的相关平台（如公众号/服务号、小程序、app等）进一步核查，如涉及相关经营主体或IP地址，可通过企查查/爱企查等查看主体是否经营异常，如IP地址为境外通常大概率为涉诈；

（3）通过互联网平台如小红书、抖音、ai工具等搜索是否有相关诈骗套路分享，如有，基本可判断为涉诈与大致套路类型。
